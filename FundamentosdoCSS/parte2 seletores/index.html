Bem-vindo

No CSS nós possuímos os seletores por ID

  • Seletor por TipoBusca elementos por uma tag HTML
  • Seletor por ID (#)Busca eleemntos através do atribuito
  • Seletor por Classe(.):Busca elementos através de atributo "class"
  • Seletor UniversalSeleciona todos os elementos HTML
  • Seletor de AtribuitoSeleciona elementos que possuem um atribuito
